# Element component
> A higher-order component (HOC) used by the other components

## Usage
```js
import {Element} from '@nimbletank/react-components'
// or
import Element from '@nimbletank/react-components/dist/Element'
```

```jsx
// Usage:
<Element tag="h2">Foobar</Element>
// Output:
<h2>Foobar</h2>
```
